          # 设备出厂时是否否激活

          # 已激活

          设备在出厂时，如果已经激活的情况下，可直接进行关联车辆

          # 未激活

          设备在出厂时，如果未激活的情况下，需要您手动激活并关联车辆；同时平台会在右下角弹出提示通知，可点击现在激活与车辆进行关联
