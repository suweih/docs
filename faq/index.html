            # Frequently Asked Questions

            How do I set up notification?

            A.Go to the Notification feature, add + notification on the right top corner. Name the notification, select the alarm condition with value and confirm notification mode. Lastly add your email to receive the notification/alarm and bind all wanted vehicles.

            How do I share a location?

            A.Go to the Sharing feature, add + on the right top corner, add - shared vehicle in the left bottom corner and then select the vehicle required. Lastly add the expiration (duration) date to generate the sharing feature.

            How do I set up a user?

            A.Go to the User feature, add + new user right top corner. Create user name and password. Select the vehicles or list that this user requires to track and save.

            How do I assign a unit or device to a vehicle?

            A.Go to the Vehicle feature; add + vehicle on the left top corner, name the vehicle, select asset type by the given drop down and select the device ID (IMEI #)

            How do I set up a Geofence?

            A.Go to the Landmark feature; add +right top corner, input the address, adjust polygon as needed, name the landmark and select if it needs to be show on the map or not. And save the new landmark created.
